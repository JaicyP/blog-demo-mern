# blog-demo-mern
FRONT END DEVELOPMENT:
    1.Reactjs
    2.Material UI
BACK END DEVELOPMENT:
    1.Node.js
    2.Expressjs
DATABASE:
    MongoDB

Steps to run:
1. To run the client side
        cd client
        npm start    in the terminal.
2. To run the server side
        cd server
        npm start    in the terminal.
3. Connect to the MongoDB Atlas before doing the above steps
